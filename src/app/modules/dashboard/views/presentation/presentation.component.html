CINEMALAND

Es una aplicación web para administrar un cine y reservar entradas en un cine. El publico que se busca captar con esta aplicación son los clientes del cine como también facilitar el manejo del cine. Aunque el cliente no necesita registrarse, en el caso de que se registre contara con funcionalidades extra como por ejemplo historial de entradas.

Funcionalidades

  • Administrador
    • Ver, crear, modificar y borrar películas, funciones, directores, géneros, usuarios y mensajes.
  • Cliente no registrado.
    • Reservar entradas.
    • Buscar películas por nombre o genero.
  • Cliente registrado
    • Reservar entradas.
    • Buscar películas por nombre o genero.
    • Ver historial de entradas.
    • Autocompletado de datos al reservar.
    • Cambiar la contraseña.

Repositorio Frontend

Frontend realizado en TypeScript con Angular

  • Subido en Vercel
  • Framework
    • Angular 14
  • Dependencias
    • Bootstrap y ng-bootstrap
    • Rxjs
  • Api externas utilizadas
    • YouTube Data API de Google
    • Movie Database API para puntuaciones

Repositorio Backend

Backend realizado en Javascript con Node.js

  • Subido en Heroku
  • Framework
    • Express
  • Dependencias
    • JWT
    • Bcrypt
    • Imágenes subidas a cloudinary
    • Base de datos MySQL subida en ClearDB
    • Sequelize
